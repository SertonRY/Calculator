# Calculator

实现了基本计算功能的Windows程序。

使用Qt Creator以及C++语言编写。

具体的学习记录可以在[CSDN](https://blog.csdn.net/weixin_42588702)上查看。

源代码在src目录下，你可以直接将整个文件夹用Qt Creator打开。

可执行文件在bin目录下。

本项目的代码仅为了学习和锻炼编程能力，主要是为了完成江苏科技大学的软件实习的要求。
